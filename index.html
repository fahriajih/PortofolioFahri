§01 — Siapa Saya

Saya Muhammad Fahri Akbar, siswa kelas XI-1 TJKT SMK Wikrama Bogor (NIS: 12410510). Saya sangat passionate di dunia Linux dan infrastruktur IT. Mulai dari Linux dasar (Ubuntu/Debian), saya berkembang ke administrasi server (DNS, DHCP, FTP, Samba, Mail Server, Web Server), monitoring stack (Prometheus + Grafana, Zabbix), automation (Ansible), cloud computing (AWS EC2, RDS, S3, Auto Scaling), hingga firewall enterprise (MikroTik, OPNsense). Saya percaya bahwa infrastruktur yang kuat adalah fondasi dari aplikasi yang handal.
